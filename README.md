# SupplySync

SupplySync is a full-stack inventory management and order tracking system built with React and Flask.

## Features

- User authentication with JWT
- Product management
- Customer management
- Order management
- Inventory tracking
- Dashboard analytics
- Low stock alerts
- Top selling products visualization

## Tech Stack

Frontend
- React
- Material UI
- Vite

Backend
- Flask
- SQLAlchemy
- JWT Authentication

Database
- SQLite

## Screens

- Dashboard analytics
- Products CRUD
- Customers CRUD
- Order creation
- Inventory monitoring
